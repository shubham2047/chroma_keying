**run using**
```
$ python app.py
```

1. Select a rectangular color patch using mouse.
2. To enable chroma keying toggle "Enable Croma keying" slider.
3. Set "Tolerance a" and "Tolerance b" as per your need.
4. press "spacebar" to pause video.
5. press "esc" to exit.
